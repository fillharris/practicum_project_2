# Practicum Coffee Shop

This is the second project of the Software Engineer program at Practicum by Yandex. It was created using HTML and CSS, based on the design brief.

## Project features

- Semantic HTML5
- Flexbox
- Positioning
- Flat BEM
- A custom form
- CSS animation and transform

## Plan on improving the project

- Embed a map with location and directions
- Add additional contact resources
- Embed a social media feed with reviews
